Usage

DynDns53 is a dynamic DNS tool running on your browser that doesn't require any software installation. To be able to use DynDns53, you'd need an AWS account.

AWS Setup

  1. Create an IAM account with Route 53 access. The IAM user must have rights to run route53:ListResourceRecordSets and route53:ChangeResourceRecordSets actions. You can use the sample policy below:
  2. Create at least one A record for the domain you want to update.

DynsDns53 Setup

  1. Enter your AWS access and secret key.
  2. Add the domain(s) you want to update to the list
  3. It's highly encouraged to test the settings by pressing Update IP now button
  4. Press Start auto-update button. As long as you keep the window open in your browser the domains will be updated with your current external IP address.

Please note, your keys never leave your browser. Everything is stored locally. If you switch browsers or go incognito, you'll have to re-enter the settings.
